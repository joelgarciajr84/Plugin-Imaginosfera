Plugin para Personalizar o WordPress da Imaginosfera

== Description ==

Personalize o wp-admin do WordPress cara clientes Imaginosfera

== Installation ==

1. Upload the `imaginosfera` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
